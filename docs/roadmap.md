# CropFleet Roadmap

---

# Phase 1 — Field Coverage Foundations (Current)

## Goals

- Polygon-based field representation
- Lane generation inside irregular fields
- Traversal generation
- Coverage visualization
- Coverage debugging workflow

## Current Progress

- Initial lane generation implemented
- Traversal path generation under development
- Coverage visualization pipeline operational
- Polygon-based field experimentation active

## Deliverables

- Working coverage planner
- Coverage visualizations
- Traversal outputs
- Public technical documentation

---

# Phase 2 — Traversal Optimization

## Goals

Improve traversal quality and operational realism.

## Planned Features

- Boustrophedon traversal ordering
- Turn minimization
- Curvature-aware smoothing
- Entry/exit optimization
- Coverage efficiency metrics
- Obstacle-aware routing

## Target Outcome

Efficient and executable agricultural coverage paths suitable for autonomous drone operation.

---

# Phase 3 — ROS2 and Simulation Integration

## Goals

Integrate the coverage pipeline into a robotics simulation environment.

## Planned Features

- ROS2 node architecture
- Mission message interfaces
- Coverage planner integration
- Gazebo simulation support
- PX4 integration
- Simulated mission execution

## Target Outcome

End-to-end simulation workflow for autonomous agricultural mission testing.

---

# Phase 4 — Multi-Drone Coordination

## Goals

Expand from single-drone coverage to decentralized multi-UAV coordination.

## Planned Features

- Multi-agent field partitioning
- Distributed task allocation
- Swarm coordination
- Conflict resolution
- Inter-drone communication
- Battery-aware mission balancing

## Target Outcome

Collaborative autonomous field coverage using multiple UAVs.

---

# Phase 5 — DroneHouse Integration

## Goals

Connect CropFleet with autonomous drone infrastructure systems.

## Planned Features

- Autonomous docking
- Charging workflows
- Fleet scheduling
- Deployment coordination
- Maintenance tracking
- Mission lifecycle management

## Target Outcome

Integrated agricultural drone fleet ecosystem.

---

# Phase 6 — Real-World Field Deployment

## Goals

Transition from simulation to operational testing.

## Planned Features

- GPS-based field execution
- Sensor integration
- Real telemetry pipelines
- Operational mission testing
- Agricultural workflow validation

## Target Outcome

Field-tested autonomous agricultural coverage framework.

---

# Long-Term Vision

CropFleet aims to evolve into a modular autonomous agricultural drone ecosystem combining:

- Coverage planning
- Autonomous navigation
- Multi-UAV coordination
- Infrastructure integration
- Agricultural mission automation

The project is intended as both a robotics engineering exploration platform and a long-term autonomous systems development initiative.
